Новые опции Easy Viewer

Добавлены новые возможности для различных настроек. Для того, чтобы увидеть, какие настройки доступны, откройте список Options... (см. рисунок 1 ниже).

Рисунок 1. Положение новых опций.

 

В открывшемся списке вы можете выбрать одну из четырёх настроек: настройку добавления события (Event Adding), настройку фильтрации (Filtering), настройку отображения CSD (CSD Displaying) и настройку вычитания среднего из каналов (Average Subtraction). См. рисунок 2.

Рисунок 2. Содержимое списка опций.

 

 

Event Adding

Теперь рассмотрим настройки по отдельности, начиная с Event Adding. Окно настроек изображено на рисунке 3. Эти настройки позволяют определить, каким образом будет добавляться событие при нажатии на кнопку Add Event.

Ранее был доступен только свободный режим (freehand), при котором событие добавлялось простым кликом мыши на графике. Теперь есть также режим locked, который в выбранном кликом участке фиксирует время события на локальном экстремуме максимуме или минимуме. Чтобы выбрать режим, откройте список напротив надписи Detection Mode.

Для корректной работы режима locked требуется также верно определить номер канала, к которому будет привязано событие. Для этого выберите номер напротив надписи Channel Number.

Полярность экстремума определяется выбором напротив Polarity. Если выбран вариант negative, то будет искаться локальный минимум, а если positive локальный максимум соответственно.

Размер временного окна, внутри которого будет искаться экстремум, указывается напротив надписи Time Window (ms) в миллисекундах.

И для того чтобы настройки сохранились, нажмите Save. Внимание, эти настройки применяются для всей программы, а не для отдельно взятого эксперимента.

 

 

Рисунок 3. Настройки для добавления события после нажатия кнопки Add Event.

Signal Filtering

Можно отсечь низкие или высокие частоты, применяя фильтрацию на выбранных каналах. Каналы, которые будут фильтроваться, выбираются из списка (см. рисунок 4).

Есть три режима фильтрации, которые можно выбрать из списка справа: highpass для того, чтобы оставить высокие частоты, lowpass для пропуска низких частот и bandpass для пропуска частот в определённом диапазоне.

Под списком режима фильтрации находятся два окошка для левой и правой границ частот. В случае выбора highpass будет работать только левая граница, при lowpass правая, а при bandpass обе.

В окошке Filter Order указывается порядок фильтра. Чем больше число, тем более строго он работает. Фильтр высокого порядка обеспечивает лучшую избирательность частот. Однако увеличение порядка фильтра увеличивает его сложность и требования к вычислениям.

После выбора параметров можно увидеть работу фильтра на частотной характеристике ниже. Для этого нажмите кнопку Check Filtration. В примере на рисунке 4 видно, как highpass фильтр оставил частоты выше выбранных 10 Гц.

Только при нажатии кнопки Apply сохранятся настройки фильтрации. Внимание, эти настройки сохраняются отдельно для каждого отдельно взятого эксперимента.

Рисунок 4. Настройка фильтрации сигналов.

Average Subtraction

Настройки вычитания среднего позволяют указать, из каких каналов можно вычесть общую среднюю составляющую. По умолчанию ни один канал не выбран для этого. Пример выбранных каналов на рисунке 5.

При нажатии кнопки Apply сохранятся настройки вычитания среднего. Внимание, эти настройки сохраняются отдельно для каждого отдельно взятого эксперимента.

Рисунок 5. Выбор каналов для вычитания среднего.

CSD Displaying

Настройки отображения CSD позволяют указать, какие каналы учитываются при расчете CSD. По умолчанию все каналы выбраны для этого. Но некоторые каналы, как, например, каналы для клампа, имеют отдельную природу, и если они выбраны, исказят CSD, поэтому их стоит исключить. Однако, по умолчанию все каналы выбраны для этой настройки. Пример выбранных каналов на рисунке 6.

При нажатии кнопки Apply сохранятся настройки отображения CSD. Внимание, эти настройки сохраняются отдельно для каждого отдельно взятого эксперимента.

Рисунок 6. Выбор каналов для отображения CSD.
